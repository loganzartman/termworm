# TermWorm
*a [pytermfx][pytermfx] demo*

![TermWorm Screenshot](promo.png?raw=true "Gameplay")

## Dependencies
0. Unix environment
1. Python 3
2. pytermfx (`pip install pytermfx`)

## Playing
* `python termworm.py`
* Use arrow keys to turn
* Collect food dots
* Press `q` to quit

[pytermfx]: https://github.com/loganzartman/pytermfx
